# 项目概述 python485

本项目是一个基于 Python Django 和 Vue3 的仓库管理系统案例源码，它集成了多种功能模块，旨在提高仓库管理的效率与准确性。

## 技术栈

- 后端：Python Django
- 前端：Vue3

## 功能模块

- 商品管理
- 库存管理
- 订单管理
- 用户权限控制

## 系统角色

- 系统管理员
- 仓库管理员
- 普通用户

## 运行环境

- Python 环境推荐：3.8及以上版本
- Django 版本：2.2及以上
- Node.js 环境：建议版本 12.0 以上
- 数据库：PostgreSQL 或 MySQL

## 部署步骤

1. 环境准备：安装 Python、Node.js 和数据库
2. 克隆代码仓库
3. 安装依赖：后端使用 `pip` 安装 Django 等依赖；前端使用 `npm` 安装相关依赖
4. 数据库配置：根据所选数据库，配置数据库连接
5. 运行后端：使用 Django 提供的命令启停服务
6. 前端构建：执行构建命令，生成静态资源
7. 部署：将构建后的前端资源放置到指定的静态文件目录

## 目录结构

```
project/
│
├── backend/         # 后端 Django 项目目录
│   ├── apps/        # 自定义 Django 应用目录
│   ├── manage.py    # Django 项目管理脚本
│   └── ...
└── frontend/        # 前端 Vue3 项目目录
    ├── node_modules/
    ├── public/
    ├── src/
    ├── package.json
    └── ...
```

## 核心亮点

- 双端分离架构，便于维护与扩展
- 细致的用户权限控制，保证系统安全
- 灵活的数据库设计，支持多种存储方案
- 基于Django的后端，提供稳定的业务逻辑处理能力
- 完善的文档支持，方便用户学习和二次开发
